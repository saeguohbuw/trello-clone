# Taskmane (Trello Clone)

Веб-приложение для управления задачами в формате kanban-доски.

## 🚀 О проекте

Проект представляет собой систему управления задачами, где пользователь может создавать доски, списки и карточки, а также управлять ими через удобный интерфейс с поддержкой drag & drop.

Реализована работа с серверными данными, динамическая загрузка контента и адаптивный интерфейс.

## 🛠️ Стек

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- React Query (работа с серверными данными)
- Zustand (управление состоянием)
- Zod + React Hook Form (валидация форм)
- Clerk (аутентификация)

## ⚙️ Основной функционал

- Работа с kanban-досками
- Drag & drop взаимодействие
- Динамическая загрузка данных
- Интеграция с backend API
- Генерация метаданных страниц (SEO)
- Авторизация пользователей

## 🧠 Что реализовано

- Архитектура frontend-приложения
- Реализация компонентов и страниц на React / Next.js
- Работа с серверными данными
- Обработка состояний (loading / error)
- Интеграция с API
- Настройка SSR и маршрутизации
